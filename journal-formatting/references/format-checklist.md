# Journal Formatting Checklist

Use this checklist as the baseline. Replace generic checks with exact journal requirements once official guidance or a template is available.

## Requirement Table

For each requirement, capture:

- Area: title page, abstract, main text, figures, tables, references, declarations, supplement, files.
- Exact journal requirement.
- Source URL/file/template and access date.
- Manuscript status: compliant, changed, missing, unclear, not applicable.
- Action taken or user input needed.

## Manuscript Structure

- Article type matches the journal category.
- Title length, running title, and subtitle rules are satisfied.
- Author names, affiliations, correspondence details, ORCID, equal contribution, consortium/group authorship, and present addresses are formatted correctly.
- Abstract is structured or unstructured as required and within word limit.
- Keywords count and style match the journal.
- Headings use allowed levels and naming.
- Main text order follows journal rules.
- Abbreviations are introduced once and used consistently.
- Units, gene/protein notation, species names, statistics, and nomenclature follow field norms and journal instructions.

## Page Layout

- Page size, margins, font, font size, line spacing, paragraph spacing, indentation, alignment, and columns match instructions.
- Page numbers and line numbers are present when required.
- Tables and figures fit within page margins in review copy.
- Section breaks, headers, footers, and title-page separation are correct.

## Figures and Tables

- Every figure/table is cited in order.
- Numbering is consecutive and matches captions and callouts.
- Captions include required definitions, statistics, sample sizes, scale bars, permissions, and abbreviations.
- Journal rules for embedded versus separate figure files are followed.
- Figure file names, panel labels, resolution, color mode, and acceptable formats are recorded.
- Tables are editable when required, not screenshots.
- Supplementary figures/tables use journal naming conventions.

## References

- In-text citation style matches the journal: numbered, author-date, superscript, bracketed, or footnote.
- Reference list order matches the citation system.
- Reference punctuation, author truncation, article titles, journal abbreviations, volume/issue/pages, DOI, PMID, arXiv, preprint labels, and accessed dates follow the target style.
- All cited references appear in the list and all listed references are cited.
- Uncertain references are flagged instead of silently normalized.

## Declarations and Submission Statements

- Funding.
- Competing interests or conflict of interest.
- Author contributions.
- Acknowledgements.
- Ethics approval, consent to participate, consent for publication.
- Data availability.
- Code availability.
- Materials availability.
- Clinical trial registration.
- Reporting guidelines and checklists.
- Permissions for reused material.

## Final Audit Report

Include:

- Journal and article type.
- Guideline sources used.
- Files delivered.
- Summary of changes.
- Compliance table.
- Missing information requiring user action.
- Items not verified, with reason.
