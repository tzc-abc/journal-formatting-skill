# Nature Portfolio Pack

## Use For

Nature, Nature Communications, Scientific Reports, Nature Materials, Nature Energy, Nature Biomedical Engineering, and other Nature Portfolio journals.

## Official Sources To Verify

- Nature Portfolio For Authors: https://www.nature.com/nature-portfolio/for-authors
- Target journal "Submission Guidelines" or "For Authors" page.
- Reporting summaries, editorial policies, data availability, code availability, and competing interests pages where applicable.

## Default Manuscript Map

- Title.
- Author list and affiliations.
- Corresponding author details.
- Abstract.
- Main text with journal-appropriate headings.
- Methods or Methods-style section when required by journal/article type.
- References.
- Acknowledgements.
- Author contributions.
- Competing interests.
- Data availability.
- Code availability when relevant.
- Additional information or supplementary information statement.

## Formatting Checks

- Confirm article type before applying length, section, and display-item limits.
- Check whether the journal uses numbered references and whether titles are included in references.
- Verify whether Methods appear in the main file, after references, or as a separate section.
- Check requirements for reporting summaries, source data, data availability, code availability, and ethics statements.
- Verify figure file formats, resolution, color, panel labels, and supplementary figure naming.

## Do Not Assume

- Do not reuse Nature flagship limits for Nature Communications or Scientific Reports.
- Do not invent word counts, figure counts, reference limits, or reporting-summary requirements.
- Do not fabricate data availability, code availability, ethics approval, or competing-interest statements.
