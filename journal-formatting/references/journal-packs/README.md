# Journal Packs

Use these packs as routing guides for mainstream journals and publishers. They are conservative templates: they identify likely manuscript sections, submission declarations, and formatting checks, but they do not replace current official author guidelines.

## How to Use

1. Match the target journal or publisher to one pack below.
2. Read the pack before building the requirement table.
3. Open the official journal-specific author guideline page and verify all exact limits.
4. Record the official source and access date in the audit report.
5. If the official page conflicts with a pack, follow the official page.

## Pack Index

- `nature-portfolio.md`: Nature, Nature Communications, Scientific Reports, and Nature Portfolio journals.
- `science-aaas.md`: Science, Science Advances, Science Translational Medicine, Science Signaling, and AAAS journals.
- `cell-press.md`: Cell, Cell Reports, Matter, iScience, and Cell Press journals.
- `wiley-materials.md`: Advanced Materials, Advanced Functional Materials, Angewandte Chemie, Small, and Wiley-VCH journals.
- `acs.md`: ACS journals such as JACS, ACS Nano, ACS Applied Materials & Interfaces, and ACS Energy Letters.
- `elsevier.md`: Elsevier journals using Guide for Authors and Editorial Manager.
- `springer-nature-bmc.md`: Springer, Springer Nature, BMC, and Palgrave journals.
- `ieee.md`: IEEE journals, transactions, magazines, and conference-style articles.
- `mdpi.md`: MDPI journals.
- `plos.md`: PLOS journals.

## Shared Rule

Never invent exact word counts, figure limits, reference limits, open-access statements, ethics wording, or article-type constraints. If the user provides no official guide and the current page cannot be accessed, create a conservative draft and mark exact requirements as "verify".
