# Elsevier Pack

## Use For

Elsevier journals using "Guide for Authors", Editorial Manager, or journal-specific submission systems.

## Official Sources To Verify

- Elsevier author policies and guidelines: https://www.elsevier.com/researcher/author/policies-and-guidelines
- Target journal "Guide for Authors" page on Elsevier.
- Journal-specific artwork, reference style, data statement, and ethics requirements.

## Default Manuscript Map

- Title page.
- Highlights when required.
- Graphical abstract when required.
- Abstract.
- Keywords.
- Main text, often Introduction, Materials and Methods, Results, Discussion, Conclusions.
- Figure and table captions.
- Acknowledgements.
- Funding.
- Declaration of competing interest.
- CRediT author statement when required.
- Data availability statement when required.
- References.

## Formatting Checks

- Verify whether the target journal uses "Your Paper Your Way" or strict initial submission formatting.
- Check if highlights, graphical abstract, CRediT, declaration of competing interest, and data availability are mandatory.
- Confirm reference style, figure file requirements, line numbering, and separate title page rules.
- Check whether tables and figures should be embedded or uploaded separately.

## Do Not Assume

- Do not treat Elsevier-wide guidance as journal-specific rules.
- Do not invent Highlights, Graphical Abstract, CRediT roles, declarations, or data statements.
- Do not force IMRaD if the journal uses a different discipline-specific structure.
